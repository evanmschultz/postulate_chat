This notebook covers how to retrieve documents from Google Drive. By default, the GoogleDriveRetriever expects the credentials.json file to be ~/.credentials/credentials.json, but this is configurable using the GOOGLE_ACCOUNT_FILE environment variable. The location of token.json use the same directory (or use the parameter token_path). Note that token.json will be created automatically the first time you use the retriever. GoogleDriveRetriever can retrieve a selection of files with some requests.  By default, If you use a folder_id, all the files inside this folder can be retrieved to Document. You can obtain your folder and document id from the URL: The special value root is for your personal home. By default, all files with these mime-type can be converted to Document. It's possible to update or customize this. See the documentation of GDriveRetriever. But, the corresponding packages must be installed. You can customize the criteria to select the files.
