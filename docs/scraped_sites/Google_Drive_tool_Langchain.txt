The location of token.json use the same directory (or use the parameter token_path). Note that token.json will be created automatically the first time you use the tool. GoogleDriveSearchTool can retrieve a selection of files with some requests.  By default, If you use a folder_id, all the files inside this folder can be retrieved to Document, if the name match the query.
